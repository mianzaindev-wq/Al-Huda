# Tests

## Unit Tests for Al-Huda Islamic AI Assistant

This directory contains test cases for the project.

### Test Structure:
- `test_*.py` files for unit tests
- `conftest.py` for pytest configuration
- Mock data and fixtures for testing

### Running Tests:
```bash
pytest tests/ -v
```

### Test Coverage:
- API routes
- Service functions
- Vector database operations
- Document processing
- Chat memory management

### Note:
Add test files as needed for new features and services.
